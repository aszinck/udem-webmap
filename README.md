# udem-webmap

This is the repository for the webmap of U-DEM. 

## Purpose

The purpose of this repository is to visualize U-DEM and make the results more interactive. It will be easy to see the added detail of U-DEM and its clear advantages. 

## Data

As a start I will visualize the RGI regions of interest using the RGIv7 dataset. Furthermore, I will take one year and/or season where I will visualise:

1. CryoSat-2 swath elevations (to be added later)
2. Sentinel-1 backscatter
3. ArcticDEM strips
4. U-DEM
